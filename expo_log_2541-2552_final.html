39 ข้อ · วิธีทำทีละขั้น · Concept & Careless Note · เหมาะสำหรับเตรียมสอบแพทย์ / วิศวะ

🧠 Cheat Sheet — เทคนิคสรุป Expo & Log สำหรับสอบเข้าแพทย์ / วิศวะ

1. สมการเอกซ์โพเนนเชียล — ทำให้ฐานเท่ากัน หรือตั้ง \(t = a^x\) (จำ: \(t > 0\) เสมอ) แล้วแก้เป็นสมการพหุนาม
2. สมการลอการิทึม — ต้องเช็ค ตัวตั้ง > 0 และ ฐาน > 0, ≠ 1 ทุกครั้ง ก่อนและหลังแก้
3. อสมการฐาน < 1 — ต้องกลับเครื่องหมายอสมการทุกครั้ง (\(0 < a < 1 \Rightarrow \log_a\) เป็นฟังก์ชันลด)
4. เปลี่ยนฐาน — \(\log_a b = \dfrac{\log_c b}{\log_c a}\), \(\log_a b \cdot \log_b a = 1\), \(\log_{a^k} b = \tfrac{1}{k}\log_a b\)
5. อัตลักษณ์สำคัญ — \(a^{\log_a x} = x\), \(\log_a a^n = n\), \(9^{\log_3 x} = x^2\), \(\log_{1/a} x = -\log_a x\)
6. แยกตัวประกอบ Expo — \((A^x - p)(B^x - q) = 0\) → แก้แต่ละตัวประกอบ (ใช้บ่อยมากในข้อสอบ Ent)
7. Log ซ้อนชั้น — ถอดจากนอกสุดเข้าในทีละชั้น ระวังเงื่อนไขโดเมนแต่ละชั้น

⚠️ จุดพลาดบ่อย (Careless & Wrong Concept)

กับดักสิ่งที่ถูกต้องประเภทความผิด
ตั้ง \(t=a^x\) แล้วลืมบังคับ \(t>0\)ตัดคำตอบที่ได้ \(t\leq0\) ทิ้งเสมอWrong Concept
ฐาน \(<1\) แล้วไม่กลับเครื่องหมายเขียนกำกับไว้ว่า "ฐาน<1 → กลับทิศ"Careless / Wrong Concept
ลืมตรวจโดเมนหลังแก้สมการ logแทนค่ากลับทุกครั้งWrong Concept
\(\log(a+b) = \log a + \log b\)\(\log(ab) = \log a + \log b\) เท่านั้นWrong Concept
สมการ \(a^{f(x)} = a^{g(x)}\) ตัดฐาน เมื่อ \(a=1\)\(1^{anything}=1\) เสมอ ต้องระวัง \(n=1\) เป็นคำตอบพิเศษWrong Concept
อสมการ \((4^x-2)\log(\cdot)>0\) มองข้ามกรณี log=0ตรวจว่า \(x=0\) ทำให้ log=0 หรือเปล่า → ตัดออกCareless
Log ซ้อนชั้น: ลืมเงื่อนไขโดเมนชั้นในทุกชั้นต้องมีอาร์กิวเมนต์ > 0Wrong Concept
1 ต.ค. 2541 สมการลอการิทึม ★★★★☆ 🟢 สมการล็อก
จงหาเซตคำตอบของ \(\log_2(x^2-x-4)^2 = \log_{0.1}(0.01)\) แล้วตอบว่าเป็นสับเซตของช่วงใด
💡 แนวคิด
คำนวณค่าด้านขวาให้เป็นตัวเลขก่อน โดยใช้ \(\log_{a^k}x = \tfrac{1}{k}\log_a x\) แล้วถอด log ออกพร้อมแยกกรณี \(\pm\)
ชั้นที่ 1 คำนวณค่า RHS

\(\log_{0.1}(0.01) = \log_{10^{-1}}(10^{-2}) = \dfrac{-2}{-1} = 2\)

ดังนั้น สมการกลายเป็น \(\log_2(x^2-x-4)^2 = 2\)
ชั้นที่ 2 ถอด log และแยกกรณี

\((x^2-x-4)^2 = 4 \Rightarrow x^2-x-4 = \pm2\)

กรณี +: \(x^2-x-6=0 \Rightarrow x=3\) หรือ \(x=-2\)

กรณี −: \(x^2-x-2=0 \Rightarrow x=2\) หรือ \(x=-1\)

ชั้นที่ 3 ตรวจโดเมน
ต้องการ \((x^2-x-4)^2 > 0\) ซึ่งเป็นจริงสำหรับทุกคำตอบข้างต้น
เซตคำตอบ \(= \{-2,-1,2,3\} \subseteq [-3,3]\)
⚠️ Careless Point
อย่าลืมว่าถอด \(\sqrt{(\cdot)^2}\) ได้ \(\pm\) — ต้องแยกทั้งสองกรณีเสมอ
ตอบข้อ 4 — \(\{-2,-1,2,3\}\subseteq[-3,3]\)
2 ต.ค. 2541 โดเมน / เรนจ์ ★★★☆☆ 🔴 โจทย์ประยุกต์
กำหนด \(f=\{(x,y)\mid y=\log(x+1)+\log(x+2)-\log(4-x^2)\}\) และ \(g=\{(x,y)\mid y=2^{x}-1,\ x\ge0\}\) จงหา \(D_f\cap R_g\) ว่าเป็นสับเซตของข้อใด
💡 แนวคิด
ก่อนอื่นหา \(D_f\) โดยต้องการให้ทุก argument ของ log เป็นบวก จากนั้นหา \(R_g\) จากช่วง \(x\ge0\) แล้วเอา intersection
ชั้นที่ 1 หาโดเมนของ \(f\)
ต้องการ\(x+1>0\)และ\(x+2>0\)และ\(4-x^2>0\)
นั้นคือ\(x>-1\)\(x>-2\)\(-2<x<2\)
ดังนั้น \(D_f = (-1,2)\)
ชั้นที่ 2 หาเรนจ์ของ \(g\)
\(y = 2^x - 1\) เพิ่ม, \(x\ge0 \Rightarrow y\ge 2^0-1=0\) ดังนั้น \(R_g = [0,\infty)\)
ชั้นที่ 3 หา intersection
\(D_f\cap R_g = (-1,2)\cap[0,\infty) = [0,2) \subseteq (0.5,2.5)\)
ตอบข้อ 2
3 มี.ค. 2542 สมบัติลอการิทึม ★★★☆☆ 🟣 สมบัติ/ทฤษฎีบท
\(\log_{10}\!\sqrt{28}-\log_{1/10}\!\sqrt{325}+\log_{1/100}91\) มีค่าเท่าใด
💡 แนวคิด
แปลงฐาน log ทุกพจน์ให้เป็นฐาน 10 โดยใช้ \(\log_{a^k}x = \frac{1}{k}\log_a x\) แล้วรวมพจน์
ชั้นที่ 1 แปลงฐาน

\(\log_{1/10}\!\sqrt{325} = -\tfrac{1}{2}\log_{10}325\), \(\quad\log_{1/100}91 = -\tfrac{1}{2}\log_{10}91\)

ชั้นที่ 2 รวมพจน์

\(\tfrac{1}{2}\log 28 + \tfrac{1}{2}\log 325 - \tfrac{1}{2}\log 91 = \tfrac{1}{2}\log\dfrac{28\times325}{91} = \tfrac{1}{2}\log 100 = 1\)

💎 เทคนิค
รวม log ก่อนคำนวณ — พยายามให้ argument ภายใน log ออกมาเป็นกำลังของ 10 เสมอ
ตอบ1
4 มี.ค. 2542 ฟังก์ชันขั้นบันได / โดเมน-เรนจ์ ★★★☆☆ 🔴 โจทย์ประยุกต์
กำหนด \(f(x)=0\) เมื่อ \(x<0\), และ \(f(x)=1\) เมื่อ \(x\ge0\); \(g=\{(x,y)\mid y=f(1-e^x),\ y>0\}\) ข้อใดถูก
💡 แนวคิด
\(y = f(1-e^x) > 0\) หมายความว่า \(f\) ต้องให้ค่าเป็น 1 → ต้องการ argument ≥ 0 → หา \(x\) ที่ทำให้ \(1-e^x \ge 0\)
ชั้นที่ 1 หาเงื่อนไข \(y > 0\)
\(f(1-e^x) = 1\) เมื่อ \(1-e^x \ge 0 \Rightarrow e^x \le 1 \Rightarrow x \le 0\)
ชั้นที่ 2 สรุป \(D_g, R_g\)
\(D_g = (-\infty,0]\), \(R_g = \{1\}\), \(R_g' = \mathbb{R}\setminus\{1\} \supseteq D_g\) → ข้อ 1 ถูก
ตอบข้อ 1
5 มี.ค. 2542 ระบบสมการ Expo + Log ★★★★★ 🔵 สมการ Expo + 🟢 สมการล็อก
ระบบสมการ: \(9^{\log_3 x}+4^{\log_2 y}=16\) และ \(\log_3 x-\log_{1/3}y=2-\log_3 2\) จงหา \(|x^2-y^2|\)
💡 แนวคิด
ใช้ \(a^{\log_a x} = x\) → \(9^{\log_3 x} = x^2\), \(4^{\log_2 y} = y^2\) แปลงสมการแรกเป็น \(x^2+y^2=16\) แล้วใช้ \(\log_{1/3}y=-\log_3 y\) แปลงสมการสอง
ชั้นที่ 1 แปลงสมการแรก

\(9^{\log_3 x} = (3^2)^{\log_3 x} = (3^{\log_3 x})^2 = x^2\), \(\quad 4^{\log_2 y} = y^2\)

ดังนั้น \(x^2 + y^2 = 16\)
ชั้นที่ 2 แปลงสมการสอง
\(\log_3 x + \log_3 y = 2 - \log_3 2 = \log_3\dfrac{9}{2}\) ดังนั้น \(xy = \dfrac{9}{2}\)
ชั้นที่ 3 คำนวณ \(|x^2-y^2|\)

\((x^2-y^2)^2 = (x^2+y^2)^2 - 4x^2y^2 = 256 - 4\!\left(\tfrac{9}{2}\right)^2 = 256-81 = 175\)

ดังนั้น \(|x^2-y^2| = \sqrt{175} = 5\sqrt{7}\)
💎 เทคนิคสำหรับแพทย์/วิศวะ
จดจำ identity \((x^2-y^2)^2=(x^2+y^2)^2-4(xy)^2\) — ใช้บ่อยมากในข้อสอบที่ให้ \(x^2+y^2\) และ \(xy\) แต่ถามเกี่ยวกับ \(x^2-y^2\)
ตอบ\(5\sqrt{7}\)
6 ต.ค. 2542 สมการเอกซ์โพเนนเชียล ★★★★★ 🔵 สมการ Expo
กำหนด \(A=\{x\in\mathbb{R}\mid 5^{3^{2x}+3}=25^{5(3^{x-1/2})}\}\) จงหาผลบวกของสมาชิกทั้งหมดใน \(A\)
💡 แนวคิด
แปลงเป็นฐาน 5 เดียวกัน เทียบเลขชี้กำลัง แล้วตั้ง \(u=3^x\) เพื่อแก้สมการกำลังสอง
ชั้นที่ 1 เทียบเลขชี้กำลังฐาน 5
\(25^{5\cdot3^{x-1/2}} = 5^{10\cdot3^{x-1/2}}\) → \(3^{2x}+3 = \dfrac{10}{\sqrt{3}}\cdot 3^x\)
ชั้นที่ 2 ตั้ง \(u = 3^x\)

\(\sqrt{3}u^2 - 10u + 3\sqrt{3} = 0\),   Disc \(= 100-36 = 64\)

\(u = \dfrac{10\pm8}{2\sqrt{3}}\) → \(u = 3^{3/2}\) หรือ \(u = 3^{-1/2}\)

ชั้นที่ 3 หาผลบวก
\(x = \tfrac{3}{2}\) หรือ \(x = -\tfrac{1}{2}\) → ผลบวก \(= 1\)
ตอบ1
7 ต.ค. 2542 สมการลอการิทึม + การนับ ★★★★☆ 🟢 สมการล็อก
ให้ \(a,b\) เป็นคำตอบของ \(\log_3 x + 6\log_x 3 = 5\) โดยที่ \(a<b\); กำหนด \(A=\{x\in\mathbb{Z}^+\mid x\in[a,b]\text{ และ }3\mid x\}\) จงหา \(n(A)\)
💡 แนวคิด
ตั้ง \(t = \log_3 x\) จะได้ \(\log_x 3 = 1/t\) แล้วแก้สมการกำลังสองใน \(t\) — จำไว้ว่า \(a,b\) คือค่าของ \(x\) ไม่ใช่ \(t\)
ชั้นที่ 1 ตั้ง \(t = \log_3 x\)

\(t + \dfrac{6}{t} = 5 \Rightarrow t^2-5t+6=0 \Rightarrow t=2\) หรือ \(t=3\)

ชั้นที่ 2 หา \(a, b\)
\(t=2 \Rightarrow x=9=a\); \(t=3 \Rightarrow x=27=b\)
ชั้นที่ 3 นับจำนวนเต็มใน \([9,27]\) ที่หาร 3 ลงตัว
\(9,12,15,18,21,24,27\) → \(n(A) = 7\)
⚠️ กับดักข้อนี้
ถ้าเข้าใจผิดว่า \(a,b\) คือค่า \(t\) (ไม่ใช่ \(x\)) จะได้ช่วง \([2,3]\) แล้วตอบ 1 ซึ่งผิด — \(a,b\) คือค่า \(x = 3^t\)
ตอบข้อ 2 — 7
8 มี.ค. 2543 สมการ Expo — แยกฐานเฉพาะ ★★★★☆ 🔵 สมการ Expo
จงหา \(x\) ที่เป็นรากของ \(2^{3x-1}\cdot6^{x}\cdot25^{5x-1} = 75^{x}\)
💡 แนวคิด
แยกทุกเลขออกเป็นฐานเฉพาะ (2, 3, 5) แล้วเปรียบเทียบเลขชี้กำลังของแต่ละฐาน
ชั้นที่ 1 แยกฐานเฉพาะ

LHS \(= 2^{4x-1}\cdot3^{x}\cdot5^{10x-2}\);   RHS \(= 3^{x}\cdot5^{2x}\)

ชั้นที่ 2 เทียบเลขชี้กำลัง
ฐาน 2:\(4x-1=0 \Rightarrow x=\tfrac{1}{4}\)
ฐาน 5:\(10x-2=2x \Rightarrow x=\tfrac{1}{4}\) ✓
ฐาน 3:เลขชี้กำลัง \(x=x\) ✓ ทุก \(x\)
ตอบ0.25
9 มี.ค. 2543 สมบัติลอการิทึม — แยกตัวประกอบ ★★★★★ 🟣 สมบัติ/ทฤษฎีบท
\(\log_3\!\dfrac{3+3\sqrt{\!\sqrt{27}}}{\sqrt{3}+5\sqrt{\!\sqrt{3}}+6}\) มีค่าเท่าใด
💡 แนวคิด
ตั้ง \(u = 3^{1/4}\) แทนทุกพจน์ (เพราะ \(\sqrt{\sqrt{3}}=3^{1/4}\), \(\sqrt{\sqrt{27}}=3^{3/4}\)) แล้วแยกตัวประกอบเศษและส่วน
ชั้นที่ 1 ตั้ง \(u = 3^{1/4}\) แทน

ตัวเศษ \(= u^4+3u^3 = u^3(u+3)\)

ตัวส่วน \(= u^2+5u+6 = (u+2)(u+3)\)

ชั้นที่ 2 ลดรูปและหา log

\(\dfrac{u^3(u+3)}{(u+2)(u+3)} = \dfrac{u^3}{u+2} = \dfrac{3^{3/4}}{3^{1/4}+2}\)

\(\log_3\dfrac{3^{3/4}}{3^{1/4}+2} = \tfrac{3}{4} - \log_3(3^{1/4}+2)\) ← ตรงกับข้อ 1

💎 เทคนิค
เมื่อเห็น \(\sqrt{\sqrt{x}}\) ให้ตั้งตัวแปรแทนทันที เช่น \(u = x^{1/4}\) ช่วยให้ตัวส่วนกลายเป็นพหุนามที่แยกตัวประกอบง่าย
ตอบข้อ 1
10 ต.ค. 2543 ถูก/ผิด — Expo & Log ★★★★☆ 🔵 สมการ Expo + 🟢 สมการล็อก
กำหนด \(U=\{2n\mid n\in\mathbb{Z}^+\}\) (จำนวนคู่บวก)
ก. มี \(x\in U\) ที่สอดคล้อง \(2^{2x+3}-18(2^x)+4=0\)
ข. มี \(x\in U\) ที่สอดคล้อง \(\log_2(x+2)+\log_2(x-1)=2\)
ชั้นที่ 1 ตรวจ ก. ตั้ง \(t=2^x\)

\(8t^2-18t+4=0 \Rightarrow (4t-1)(t-2)=0 \Rightarrow t=\tfrac{1}{4}\) หรือ \(t=2\)

\(x=-2\) หรือ \(x=1\) — ทั้งคู่ไม่อยู่ใน \(U\) → ก. ผิด

ชั้นที่ 2 ตรวจ ข.

\((x+2)(x-1)=4 \Rightarrow x^2+x-6=0 \Rightarrow x=2\) (ใช้ได้) หรือ \(x=-3\) (ตัดทิ้ง)

\(x=2\in U\) → ข. ถูก

ตอบข้อ 3 (ก ผิด ข ถูก)
11 ต.ค. 2543 สมการ Log — เซต ★★★★★ 🟢 สมการล็อก
\(A=\{x\in\mathbb{R}^+\mid x^{\log_3 x^3}=9x\}\), \(B=\{x\in\mathbb{R}^+\mid \log_3 x^x = \frac{x}{3}\}\), \(C=\{ab\mid a\in A,\,b\in B\}\) — เซตใดเป็นสับเซตของ \(C\)?
💡 แนวคิด
ตั้ง \(p = \log_3 x\) แก้แต่ละเซต แล้วคำนวณ \(C\) จากการคูณ \(a \times b\)
ชั้นที่ 1 หา \(A\)

ให้ \(p=\log_3 x\): \(3p^2-p-2=0 \Rightarrow p=-\tfrac{2}{3}\) หรือ \(p=1\)

\(A = \{3^{-2/3},\;3\}\)

ชั้นที่ 2 หา \(B\)

\(x\log_3 x = \tfrac{x}{3} \Rightarrow \log_3 x = \tfrac{1}{3} \Rightarrow B = \{3^{1/3}\}\)

ชั้นที่ 3 หา \(C\)
\(C = \{3^{-2/3}\cdot3^{1/3},\;3\cdot3^{1/3}\} = \{3^{-1/3},\;3^{4/3}\}\) ← ตรงข้อ 2
ตอบข้อ 2
12 ต.ค. 2543 กราฟ Expo + เส้นตรง ★★★★☆ 🔴 กราฟฟังก์ชัน
เส้นโค้ง \(y=2^{2x}-2^{x+2}-45\) ตัดแกน \(x\) ที่จุด \(A\) และเส้นผ่านจุด \(A\) และ \(B(0,b)\) ขนานกับ \(y=(\log_3 2)x-4\) จงหา \(b\)
ชั้นที่ 1 หาจุด \(A\)

ให้ \(t=2^x\): \(t^2-4t-45=0 \Rightarrow t=9\) → \(x=\log_2 9\), \(A=(\log_2 9,0)\)

ชั้นที่ 2 ใช้ความชันหา \(b\)

ความชัน \(= -\dfrac{b}{\log_2 9} = \log_3 2\)

\(b = -\log_2 9\cdot\log_3 2 = -2\log_2 3\cdot\dfrac{1}{\log_2 3} = -2\)

💎 เทคนิค
\(\log_2 9\cdot\log_3 2 = 2\log_2 3\cdot\dfrac{\log_2 2}{\log_2 3} = 2\) — ใช้ chain rule ของ log เปลี่ยนฐาน
ตอบ\(-2\)
13 มี.ค. 2544 อสมการลอการิทึมซ้อน ★★★★★ 🟡 อสมการล็อก
ให้ \(A\) คือเซตคำตอบของ \(\log_4\!\log_3\!\log_2(x^2+2x)\le0\) จงหาจำนวนเต็มใน \(A\)
💡 แนวคิด
ถอด log ซ้อนชั้นจากนอกสุดเข้าในทีละชั้น — แต่ละชั้นต้องระวังทิศอสมการและเงื่อนไขโดเมน (argument > 0)
ชั้นที่ 1 ถอดทีละชั้น

\(\log_4(\cdot)\le0 \Rightarrow 0 < \log_3\!\log_2(x^2+2x) \le 1\)

\(\Rightarrow 1 < \log_2(x^2+2x) \le 3 \Rightarrow 2 < x^2+2x \le 8\)

ชั้นที่ 2 แก้อสมการกำลังสอง

\(A = [-4,\,-1-\sqrt{3}\,)\cup(-1+\sqrt{3},\,2]\)

(\(-1-\sqrt{3}\approx-2.73\), \(-1+\sqrt{3}\approx0.73\))

จำนวนเต็มในช่วง: \(-4,-3,1,2\) → 4 จำนวน

⚠️ Careless Point
เมื่อถอดชั้น \(\log_4(\cdot)\le0\) ต้องบังคับ argument > 0 ด้วย → ได้ \(0 < \log_3\!\log_2(\cdot) \le 1\) ไม่ใช่ \(\le1\) อย่างเดียว

ก. ถ้า \((\log a)^3=x-1\) และ \((\log b)^3=x+1\) แล้ว \(\log(a+b)=\sqrt[3]{x^2-1}\)
ข. กราฟ \(y=x^2\) และ \(y=2^x\) ตัดกัน 2 จุด
ชั้นที่ 1 ตรวจ ก.
\(\sqrt[3]{x^2-1} = (\log a)(\log b) \ne \log(a+b)\) ทั่วไป → ก. ผิด
ชั้นที่ 2 ตรวจ ข.
\(x^2=2^x\) มีคำตอบ 3 ค่า: \(x=2, x=4, x\approx-0.77\) → กราฟตัดกัน 3 จุด ไม่ใช่ 2 → ข. ผิด
⚠️ จุดพลาดบ่อย
มองข้ามจุดตัดแถวจำนวนลบ \(x\approx-0.77\) — ควรวาดกราฟคร่าวๆ เสมอ
ตอบข้อ 4 (ก ผิด ข ผิด)
38 A-NET 2552 สมการ Expo — จำนวนธรรมชาติ ★★★★☆ 🔵 สมการ Expo
\(A=\{n\in\mathbb{N}\mid n^{n^2+9}=n^{n^3-9}\}\), \(B=\{n\in\mathbb{N}\mid\log n=\log(n+1)\}\) จงหาผลบวกสมาชิกใน \(A\cup B\)
💡 แนวคิด
สมการ \(n^p=n^q\) มีคำตอบพิเศษ \(n=1\) (เพราะ \(1^k=1\) เสมอ) นอกจาก \(p=q\) — อย่าลืมตรวจกรณีนี้ด้วย
ชั้นที่ 1 หา \(A\)

\(n=1\): \(1^k=1\) ✓; กรณี \(n\ge2\): \(n^2+9=n^3-9 \Rightarrow n=3\) ✓

\(A=\{1,3\}\)

ชั้นที่ 2 หา \(B\)
\(\log n=\log(n+1)\) เป็นไปไม่ได้ → \(B=\varnothing\) → ผลบวก \(=1+3=4\)
⚠️ กับดักข้อนี้ (Wrong Concept)
ถ้าตัดฐานทันทีจะได้ \(n^2+9=n^3-9\) แล้วหา \(n=3\) เท่านั้น — แต่ลืม \(n=1\) เป็นคำตอบพิเศษ!

39 A-NET 2552 อัตลักษณ์ Log — พีทาโกรัส ★★★★★ 🟣 สมบัติ/ทฤษฎีบท
กำหนด \(a>1,\ b,c>0,\ a^2+b^2=c^2\) และ \(\log_{c+b}a+\log_{c-b}a = x\cdot(\log_{c+b}a)(\log_{c-b}a)\) จงหา \(x\)
💡 แนวคิด
หารทั้งสองข้างด้วยผลคูณ แล้วใช้ \(\frac{1}{\log_p a}=\log_a p\) เพื่อเปลี่ยนกลับ จากนั้นใช้ \(c^2-b^2=a^2\)
ชั้นที่ 1 หาร และแปลงด้วย \(\frac{1}{\log_p a}=\log_a p\)

\(x = \log_a(c-b)+\log_a(c+b) = \log_a(c^2-b^2)\)

ชั้นที่ 2 ใช้ \(a^2+b^2=c^2\)
\(c^2-b^2=a^2 \Rightarrow x=\log_a a^2 = 2\)
