🧠 สรุปเทคนิครวบยอดบทอินทิกรัล

1. กฎพื้นฐานที่ต้องจำขึ้นใจ

  • Power Rule: \(\int x^n \, dx = \dfrac{x^{n+1}}{n+1} + C\) (เมื่อ \(n \neq -1\))
  • FTC (Fundamental Theorem of Calculus): \(\int_a^b f(x)\,dx = F(b) - F(a)\) เมื่อ \(F' = f\)
  • สมการเส้นสัมผัส: \(y - f(a) = f'(a)(x-a)\)
  • ความชัน = อนุพันธ์อันดับ 1: \(f'(x)\) ให้ความชัน ณ จุด \(x\)
  • จุดวิกฤต: \(f'(x) = 0\) หรือ \(f'(x)\) ไม่มีอยู่
  • ทดสอบอนุพันธ์อันดับสอง: \(f''(c) > 0 \Rightarrow\) น้อยสุด, \(f''(c) < 0 \Rightarrow\) มากสุด

2. เทคนิค "ทอดกลับ" (Anti-differentiation) — ข้อสอบยอดฮิต

  • โจทย์ให้ ความชัน \(f'(x) = g(x)\) → อินทิเกรตเพื่อหา \(f(x)\)
  • โจทย์ให้ \(f''(x)\) → อินทิเกรตสองครั้ง หา \(f'(x)\) แล้วหา \(f(x)\)
  • ใช้ เงื่อนไขจุดผ่าน \((a, f(a))\) เพื่อหาค่าคงตัว \(C\)
  • เงื่อนไข "พื้นที่สองเท่า" / "พื้นที่เท่ากัน" → ตั้งสมการเปรียบเทียบอินทิกรัล

3. พื้นที่ (Area) vs อินทิกรัล (Integral) — อย่าสับสน!

  • อินทิกรัล \(\int_a^b f(x)\,dx\) = พื้นที่ มีเครื่องหมาย (บวก-ลบ)
  • พื้นที่ \(A = \int_a^b |f(x)|\,dx\) = พื้นที่ จริง (บวกเสมอ)
  • หาก \(f(x)\) เปลี่ยนเครื่องหมายบน \([a,b]\): แบ่งช่วง แล้วหาค่าสัมบูรณ์ในแต่ละช่วง
  • รูปเรขาคณิต: \(\int_{-r}^{r} \sqrt{r^2-x^2}\,dx = \dfrac{\pi r^2}{2}\) (ครึ่งวงกลม)

4. เทคนิคพิเศษ

  • Substitution: \(\int_a^b f(g(x))g'(x)\,dx = \int_{g(a)}^{g(b)} f(u)\,du\)
  • Integration by Parts: \(\int u\,dv = uv - \int v\,du\)
  • ฟังก์ชันคู่/คี่: \(\int_{-a}^{a} f(x)\,dx = 2\int_0^a f(x)\,dx\) (คู่), \(= 0\) (คี่)
  • Chain Rule + Integral: ถ้า \(F(x) = f(g(x))\) แล้ว \(F'(x) = f'(g(x))g'(x)\)
  • Piecewise: แบ่งการอินทิเกรตตามช่วงนิยามของแต่ละ piece

Q1 ต.ค. 41/3🟣 หาฟังก์ชันจากอนุพันธ์ + เงื่อนไขเริ่มต้น
กำหนดให้ \(f\) เป็นฟังก์ชัน ที่ \(f(2) = -1\), \(f'(1) = -3\) และ \(f''(x) = 3\) ทุกค่า \(x\) แล้ว \(f(0)\) มีค่าเป็นเท่าใด
1) 5 2) 6 3) 12 4) 15
คอนเซปต์: Anti-differentiation สองครั้ง (จาก \(f''\) → \(f'\) → \(f\)) พร้อมใช้เงื่อนไขเพื่อหาค่าคงตัว
วิธีทำ (Chain of Thought):
ขั้นที่ 1: อินทิเกรต \(f''(x) = 3\) เพื่อหา \(f'(x)\)
\[f'(x) = \int 3\,dx = 3x + C_1\]
ขั้นที่ 2: ใช้เงื่อนไข \(f'(1) = -3\) หา \(C_1\)
\[f'(1) = 3(1) + C_1 = -3 \Rightarrow C_1 = -6\]
\[\therefore f'(x) = 3x - 6\]
ขั้นที่ 3: อินทิเกรต \(f'(x)\) เพื่อหา \(f(x)\)
\[f(x) = \int (3x-6)\,dx = \dfrac{3x^2}{2} - 6x + C_2\]
ขั้นที่ 4: ใช้เงื่อนไข \(f(2) = -1\) หา \(C_2\)
\[f(2) = \dfrac{3(4)}{2} - 6(2) + C_2 = 6 - 12 + C_2 = -1\]
\[C_2 = 5\]
\[\therefore f(x) = \dfrac{3x^2}{2} - 6x + 5\]
ขั้นที่ 5: หา \(f(0)\)
\[f(0) = 0 - 0 + 5 = 5\]
\(f(0) = 5\) → ตอบ ข้อ 1
เมื่อโจทย์ให้ \(f''(x)\) และเงื่อนไข 2 จุด ให้อินทิเกรต ตามลำดับ แล้วหาค่าคงตัวทีละตัว — ไม่ต้องตั้งระบบสมการ
ระวัง! อย่าสลับ \(f'(1) = -3\) กับ \(f(1) = -3\) — เครื่องหมาย \('\) ตัวเดียวเปลี่ยนความหมายโจทย์ทั้งหมด
Q2 ต.ค. 41/3🟡 พื้นที่ใต้กราฟ/ระหว่างเส้นโค้ง
พื้นที่ปิดล้อมด้วยเส้นโค้ง \(y = x^2 - 3x + 2\) จาก \(x = 0\) ถึง \(x = 2\) เฉพาะส่วนที่อยู่เหนือแกน \(x\) เท่ากับข้อใด
1) \(\frac{3}{2}\) ตารางหน่วย 2) \(\frac{1}{6}\) ตารางหน่วย 3) \(\frac{2}{3}\) ตารางหน่วย 4) \(\frac{5}{6}\) ตารางหน่วย
คอนเซปต์: พื้นที่ "เฉพาะส่วนเหนือแกน \(x\)" — ต้องหาช่วงที่ \(f(x) \geq 0\) ก่อน แล้วอินทิเกรตเฉพาะช่วงนั้น
วิธีทำ:
ขั้นที่ 1: แยกตัวประกอบ \(y = x^2 - 3x + 2 = (x-1)(x-2)\)
ราก: \(x = 1, 2\)
ขั้นที่ 2: วิเคราะห์เครื่องหมายบน \([0, 2]\)
• ช่วง \([0, 1)\): \(x-1 < 0, x-2 < 0\) → ผลคูณบวก → \(y \geq 0\) ✓ (เหนือแกน)
• ช่วง \((1, 2)\): \(x-1 > 0, x-2 < 0\) → ผลคูณลบ → \(y < 0\) (ใต้แกน)
ขั้นที่ 3: อินทิเกรตเฉพาะส่วนเหนือแกน \(x\) คือ \([0, 1]\)
\[A = \int_0^1 (x^2 - 3x + 2)\,dx = \left[\dfrac{x^3}{3} - \dfrac{3x^2}{2} + 2x\right]_0^1\]
\[= \left(\dfrac{1}{3} - \dfrac{3}{2} + 2\right) - 0 = \dfrac{2 - 9 + 12}{6} = \dfrac{5}{6}\]
พื้นที่ \(= \dfrac{5}{6}\) ตารางหน่วย → ตอบ ข้อ 4
โจทย์ถาม "เฉพาะเหนือแกน \(x\)" → ไม่ใช่พื้นที่รวม ต้องหาช่วงที่ \(f(x) \geq 0\) ก่อนเสมอ
อย่าเพิ่งอินทิเกรต \(\int_0^2\) ตรงๆ เพราะจะได้พื้นที่สุทธิ (บวก-ลบหักล้าง) ซึ่งผิดทั้งความหมายและคำตอบ
Q3 มี.ค. 42/3🟢 ทฤษฎีบทหลักแคลคูลัส (FTC) & อินทิกรัลจำกัดเขต
ถ้า \(\theta \in \mathbb{R}\) และ \(\displaystyle\int_{\sin\theta}^{1} (4x - 3)\,dx = 0\) แล้ว \(\cos 2\theta\) เท่ากับข้อใด
1) \(0\) หรือ \(\frac{\sqrt{3}}{2}\) 2) \(0\) หรือ \(-\frac{\sqrt{3}}{2}\) 3) \(-1\) หรือ \(\frac{1}{2}\) 4) \(-1\) หรือ \(1\)
คอนเซปต์: FTC + สมการตรีโกณมิติ ใช้เอกลักษณ์ \(\cos 2\theta = 1 - 2\sin^2\theta\)
วิธีทำ:
ขั้นที่ 1: คำนวณอินทิกรัล
\[\int_{\sin\theta}^{1} (4x-3)\,dx = \left[2x^2 - 3x\right]_{\sin\theta}^{1}\]
\[= (2 - 3) - (2\sin^2\theta - 3\sin\theta) = -1 - 2\sin^2\theta + 3\sin\theta = 0\]
ขั้นที่ 2: จัดรูปเป็นสมการ
\[2\sin^2\theta - 3\sin\theta + 1 = 0\]
\[(2\sin\theta - 1)(\sin\theta - 1) = 0\]
ขั้นที่ 3: หาค่า \(\sin\theta\)
\[\sin\theta = \dfrac{1}{2} \quad \text{หรือ} \quad \sin\theta = 1\]
ขั้นที่ 4: ใช้เอกลักษณ์ \(\cos 2\theta = 1 - 2\sin^2\theta\)
• กรณี \(\sin\theta = \dfrac{1}{2}\): \(\cos 2\theta = 1 - 2\cdot\dfrac{1}{4} = \dfrac{1}{2}\)
• กรณี \(\sin\theta = 1\): \(\cos 2\theta = 1 - 2 = -1\)
\(\cos 2\theta = -1\) หรือ \(\dfrac{1}{2}\) → ตอบ ข้อ 3
เมื่อเห็น \(\cos 2\theta\) และมี \(\sin\theta\) ในโจทย์ → ใช้ \(\cos 2\theta = 1 - 2\sin^2\theta\) เปลี่ยนตัวแปรเสมอ
อย่าลืมเครื่องหมายลบตอนคำนวณ \((2 - 3) = -1\) — เครื่องหมายนี้บอกวิธีจัดสมการทั้งหมด
Q4 ต.ค. 42/3🟣 หาฟังก์ชันจากอนุพันธ์ + เงื่อนไขเริ่มต้น
ถ้า \(f\) เป็นฟังก์ชันซึ่งมีกราฟผ่านจุด \((0, 2)\) และ \(f'(x) = 3x^2 - 12x + 9\) แล้วค่าสูงสุดสัมพัทธ์ของ \(f\) เท่ากับข้อใด
1) 2 2) 3 3) 6 4) 8
คอนเซปต์: อินทิเกรต \(f'\) เพื่อหา \(f\), หาจุดวิกฤตด้วย \(f' = 0\), ตรวจด้วย \(f''\)
วิธีทำ:
ขั้นที่ 1: อินทิเกรต \(f'\)
\[f(x) = \int (3x^2 - 12x + 9)\,dx = x^3 - 6x^2 + 9x + C\]
ขั้นที่ 2: ใช้จุดผ่าน \((0, 2)\) หา \(C\)
\[f(0) = C = 2 \Rightarrow f(x) = x^3 - 6x^2 + 9x + 2\]
ขั้นที่ 3: หาจุดวิกฤต \(f'(x) = 0\)
\[3x^2 - 12x + 9 = 0 \Rightarrow 3(x-1)(x-3) = 0 \Rightarrow x = 1 \text{ หรือ } x = 3\]
ขั้นที่ 4: ทดสอบอนุพันธ์อันดับสอง
\[f''(x) = 6x - 12\]
• \(f''(1) = -6 < 0\) → สูงสุดสัมพัทธ์ที่ \(x = 1\)
• \(f''(3) = 6 > 0\) → ต่ำสุดสัมพัทธ์ที่ \(x = 3\)
ขั้นที่ 5: หาค่าสูงสุด
\[f(1) = 1 - 6 + 9 + 2 = 6\]
ค่าสูงสุดสัมพัทธ์ \(= 6\) → ตอบ ข้อ 3
ทดสอบอนุพันธ์อันดับสองเร็ว: \(f'' < 0 \Rightarrow\) max, \(f'' > 0 \Rightarrow\) min — ไม่ต้องตารางเครื่องหมาย
อย่าลืมใช้จุดผ่าน \((0, 2)\) หา \(C\) — ถ้าลืบจะได้ \(f(x)\) ไม่สมบูรณ์ และคำตอบจะผิด
Q5 ต.ค. 42/4🟢 ทฤษฎีบทหลักแคลคูลัส (FTC) & อินทิกรัลจำกัดเขต
ให้ \(F(x) = f(g(x))\) ถ้า \(g(x) = x^3 + 2x + 2\) และ \(\int F(x)\,dx = 5x^3 + 2x + c\) แล้ว ค่าของ \(f'(5)\) เท่ากับข้อใด
1) 6 2) 5 3) 4 4) 3
คอนเซปต์: FTC (อินทิเกรตกลับ) + Chain Rule: \(F'(x) = f'(g(x)) \cdot g'(x)\)
วิธีทำ:
ขั้นที่ 1: อินทิเกรตกลับเพื่อหา \(F(x)\)
\[\int F(x)\,dx = 5x^3 + 2x + c \Rightarrow F(x) = 15x^2 + 2\]
ขั้นที่ 2: ดิฟ \(F(x)\) และใช้ Chain Rule
\[F'(x) = 30x\]
\[F'(x) = f'(g(x)) \cdot g'(x) = 30x\]
ขั้นที่ 3: หา \(g'(x)\)
\[g'(x) = 3x^2 + 2\]
ขั้นที่ 4: หา \(x\) ที่ทำให้ \(g(x) = 5\) (เพื่อให้ได้ \(f'(5)\))
\[g(x) = x^3 + 2x + 2 = 5 \Rightarrow x^3 + 2x - 3 = 0\]
\[(x-1)(x^2+x+3) = 0 \Rightarrow x = 1\] (เท่านั้นที่เป็นจริง)
ขั้นที่ 5: แทนค่า \(x = 1\)
\[f'(g(1)) \cdot g'(1) = 30(1)\]
\[f'(5) \cdot (3+2) = 30 \Rightarrow f'(5) = \dfrac{30}{5} = 6\]
\(f'(5) = 6\) → ตอบ ข้อ 1
เทคนิค: โจทย์ขอ \(f'(a)\) → หา \(x_0\) ที่ทำให้ \(g(x_0) = a\) ก่อน แล้วแทนใน Chain Rule
ระวังสับสน \(F(x)\) (ฟังก์ชัน composition) กับ \(\int F(x)\,dx\) (ปฏิยานุพันธ์) — ดิฟเพื่อหา \(F\) จากอินทิกรัล
Q6 มี.ค. 43/3🔵 ปฏิยานุพันธ์ & สูตรอินทิเกรตพื้นฐาน
ให้ \(F\) เป็นปฏิยานุพันธ์ของ \(f\) โดยที่ \(f(x) = 3x^2 - 6x + 3\) ถ้า \(F(0) = -1\) และ \(F\) มีค่าสูงสุดสัมบูรณ์ในช่วง \([0, 2]\) ที่จุด \(x = c\) แล้ว \(F(c)\) เท่ากับข้อใด
1) -1 2) 0 3) 1 4) 2
คอนเซปต์: \(F' = f \geq 0 \Rightarrow F\) ไม่ลด — สูงสุดสัมบูรณ์อยู่ที่ปลายขวาของช่วง
วิธีทำ:
ขั้นที่ 1: สังเกตว่า \(f(x) = 3(x-1)^2 \geq 0\) ทุก \(x\)
ดังนั้น \(F'(x) = f(x) \geq 0\) หมายความว่า \(F\) ไม่ลด บน \([0, 2]\)
ขั้นที่ 2: สูงสุดสัมบูรณ์บน \([0, 2]\) จึงอยู่ที่ปลายขวา \(c = 2\)
ขั้นที่ 3: หา \(F(x)\)
\[F(x) = \int (3x^2 - 6x + 3)\,dx = x^3 - 3x^2 + 3x + C\]
\[F(0) = C = -1\]
\[F(x) = x^3 - 3x^2 + 3x - 1 = (x-1)^3\]
ขั้นที่ 4: หา \(F(2)\)
\[F(2) = (2-1)^3 = 1\]
\(F(c) = F(2) = 1\) → ตอบ ข้อ 3
เทคนิคลัด: ถ้า \(f(x) \geq 0\) ทั้งช่วง → \(F\) ไม่ลด → max abs = \(F(\text{ปลายขวา})\), min abs = \(F(\text{ปลายซ้าย})\)
อย่าเพิ่งตั้ง \(F' = 0\) — ต้อง วิเคราะห์เครื่องหมายของ \(f\) ก่อน เพราะ \(F' = 0\) อาจไม่ให้สูงสุดสัมบูรณ์
Q7 มี.ค. 43/3🟢 ทฤษฎีบทหลักแคลคูลัส (FTC) & อินทิกรัลจำกัดเขต
กำหนดให้ \(f\) เป็นฟังก์ชันที่มีอนุพันธ์ และ \(g(x) = (x+1)f(x)\) ถ้า \(\int g(x)\,dx = x^2 - x + c\) แล้ว \(f'(1)\) มีค่าเท่ากับข้อใด
1) \(\frac{3}{4}\) 2) \(\frac{5}{4}\) 3) \(\frac{3}{2}\) 4) \(\frac{5}{2}\)
คอนเซปต์: อินทิเกรตกลับเพื่อหา \(g\), ใช้ Product Rule \(g'(x) = f(x) + (x+1)f'(x)\)
วิธีทำ:
ขั้นที่ 1: อินทิเกรตกลับหา \(g(x)\)
\[\int g(x)\,dx = x^2 - x + c \Rightarrow g(x) = 2x - 1\]
ขั้นที่ 2: หา \(f(1)\) จาก \(g(1) = (1+1)f(1)\)
\[g(1) = 2(1) - 1 = 1 = 2f(1) \Rightarrow f(1) = \dfrac{1}{2}\]
ขั้นที่ 3: ดิฟ \(g(x) = (x+1)f(x)\) ด้วย Product Rule
\[g'(x) = f(x) + (x+1)f'(x)\]
\[g'(x) = 2\] (จาก \(g(x) = 2x - 1\))
ขั้นที่ 4: แทน \(x = 1\)
\[g'(1) = f(1) + 2f'(1) = 2\]
\[\dfrac{1}{2} + 2f'(1) = 2 \Rightarrow 2f'(1) = \dfrac{3}{2} \Rightarrow f'(1) = \dfrac{3}{4}\]
\(f'(1) = \dfrac{3}{4}\) → ตอบ ข้อ 1
Product Rule: \((uv)' = u'v + uv'\) — จำรูป \(g(x) = (x+1)f(x) \Rightarrow g'(x) = f(x) + (x+1)f'(x)\)
ระวังสับสน \(f(1)\) กับ \(f'(1)\) — โจทย์ถาม \(f'(1)\) ต้องใช้ Product Rule หา ไม่ใช่แค่แทนค่า
Q8 ต.ค. 43/ตอนที่ 1🟣 หาฟังก์ชันจากอนุพันธ์ + เงื่อนไขเริ่มต้น
ถ้าเส้นโค้ง \(y = f(x)\) ผ่านจุด \((0, 1)\) และ \((4, c)\) เมื่อ \(c\) เป็นจำนวนจริง และความชันของเส้นโค้งนี้ที่จุด \((x, y)\) ใดๆ มีค่าเท่ากับ \(\sqrt{x} - 1\) แล้ว \(c\) มีค่าเท่ากับข้อใด
1) \(\frac{4}{3}\) 2) \(\frac{7}{3}\) 3) 8 4) 9
คอนเซปต์: ความชัน = \(f'(x)\) → อินทิเกรตเพื่อหา \(f(x)\), ใช้จุดผ่านหา \(C\)
วิธีทำ:
ขั้นที่ 1: ตั้งสมการ
\[f'(x) = \sqrt{x} - 1 = x^{1/2} - 1\]
ขั้นที่ 2: อินทิเกรต
\[f(x) = \dfrac{x^{3/2}}{3/2} - x + C = \dfrac{2x^{3/2}}{3} - x + C\]
ขั้นที่ 3: ใช้ \(f(0) = 1\)
\[f(0) = 0 - 0 + C = 1 \Rightarrow C = 1\]
\[f(x) = \dfrac{2x^{3/2}}{3} - x + 1\]
ขั้นที่ 4: หา \(c = f(4)\)

กำหนดให้ \[f(x) = \begin{cases} x + 3 & ; x < -1 \\ -2x^3 & ; x \geq -1 \end{cases}\] พื้นที่ปิดล้อมด้วยกราฟของ \(f\) บนช่วง \([-4, 0]\) มีค่าเท่าใด
คอนเซปต์: Piecewise + พื้นที่ = ผลรวมของค่าสัมบูรณ์ — ต้องหาจุดที่ \(f\) เปลี่ยนเครื่องหมาย
วิธีทำ:
ขั้นที่ 1: แบ่งช่วงนิยาม
• ช่วง \([-4, -1)\): \(f(x) = x + 3\) เปลี่ยนเครื่องหมายที่ \(x = -3\)
  – \([-4, -3)\): \(f < 0\)
  – \((-3, -1)\): \(f > 0\)
• ช่วง \([-1, 0]\): \(f(x) = -2x^3\). เมื่อ \(x \in [-1, 0]\): \(x^3 \leq 0 \Rightarrow -2x^3 \geq 0\) → \(f \geq 0\)
ขั้นที่ 2: แบ่งอินทิเกรตเป็น 3 ส่วน
\[A = \int_{-4}^{-3} |x+3|\,dx + \int_{-3}^{-1} |x+3|\,dx + \int_{-1}^{0} |-2x^3|\,dx\]
\[= \int_{-4}^{-3} -(x+3)\,dx + \int_{-3}^{-1} (x+3)\,dx + \int_{-1}^{0} (-2x^3)\,dx\]
ขั้นที่ 3: คำนวณแต่ละส่วน
• \(\int_{-4}^{-3} -(x+3)\,dx = \left[-\dfrac{(x+3)^2}{2}\right]_{-4}^{-3} = 0 - \left(-\dfrac{1}{2}\right) = \dfrac{1}{2}\)
• \(\int_{-3}^{-1} (x+3)\,dx = \left[\dfrac{(x+3)^2}{2}\right]_{-3}^{-1} = \dfrac{4}{2} - 0 = 2\)
• \(\int_{-1}^{0} (-2x^3)\,dx = \left[-\dfrac{x^4}{2}\right]_{-1}^{0} = 0 - \left(-\dfrac{1}{2}\right) = \dfrac{1}{2}\)
ขั้นที่ 4: รวม
\[A = \dfrac{1}{2} + 2 + \dfrac{1}{2} = 3\]
พื้นที่ \(= 3\) ตารางหน่วย
\(\int (x+3)^n\,dx = \dfrac{(x+3)^{n+1}}{n+1}\) — ใช้ substitution \(u = x+3\) ลดรูป
อย่าลืมว่า พื้นที่ ≠ อินทิกรัล — ถ้าคำนวณเป็นอินทิกรัล (ไม่มีค่าสัมบูรณ์) จะได้ 2 ไม่ใช่ 3
Q36 A-NET พ.ศ. 2552 ข้อ 2 คะแนน🟡 พื้นที่ใต้กราฟ/ระหว่างเส้นโค้ง
จงหาพื้นที่ใต้กราฟของฟังก์ชันที่มีสมการ \(y = x^3 - 4x\) และแกน \(x\) โดยที่ \(x\) อยู่ในช่วง \([-2, 1]\) เท่ากับเท่าใด
คอนเซปต์: แยกตัวประกอบ → หาราก → แบ่งช่วงเครื่องหมาย → อินทิเกรต |f|
วิธีทำ:
ขั้นที่ 1: แยกตัวประกอบ
\[f(x) = x^3 - 4x = x(x^2 - 4) = x(x-2)(x+2)\]
ราก: \(x = -2, 0, 2\)
ขั้นที่ 2: วิเคราะห์เครื่องหมายบน \([-2, 1]\)
• ช่วง \((-2, 0)\): \(x < 0, x-2 < 0, x+2 > 0\) → ผลคูณบวก → \(f > 0\) (เหนือแกน)
• ช่วง \((0, 1)\): \(x > 0, x-2 < 0, x+2 > 0\) → ผลคูณลบ → \(f < 0\) (ใต้แกน)
ขั้นที่ 3: แบ่งอินทิเกรต
\[A = \int_{-2}^{0} (x^3 - 4x)\,dx + \int_0^1 -(x^3 - 4x)\,dx\]
ขั้นที่ 4: คำนวณ
\[\int (x^3 - 4x)\,dx = \dfrac{x^4}{4} - 2x^2\]
• \(\int_{-2}^{0} (x^3 - 4x)\,dx = 0 - \left(\dfrac{16}{4} - 2 \cdot 4\right) = 0 - (4 - 8) = 4\)
• \(\int_0^1 -(x^3 - 4x)\,dx = -\left[\dfrac{x^4}{4} - 2x^2\right]_0^1 = -\left(\dfrac{1}{4} - 2\right) = -\left(-\dfrac{7}{4}\right) = \dfrac{7}{4}\)
\[A = 4 + \dfrac{7}{4} = \dfrac{16 + 7}{4} = \dfrac{23}{4} = 5.75\]
พื้นที่ \(= \dfrac{23}{4} = 5.75\) ตารางหน่วย
พหุนามดีกรี 3 ที่มี 3 รากจริง → เครื่องหมายสลับทุกครั้งที่ข้ามราก (กฎของพหุนามดีกรีคี่)
\([x^4/4 - 2x^2]\) ที่ \(x = -2\): \(\dfrac{16}{4} - 2 \cdot 4 = 4 - 8 = -4\) → ระวังเครื่องหมาย

📚 กลุ่มเทคนิคสำคัญ

1. Anti-differentiation (ทอดกลับ) — พบใน Q1, Q4, Q5, Q7, Q8, Q13, Q15, Q17, Q18, Q20, Q21, Q22, Q30, Q31

  • โจทย์ให้ \(f'\) หรือ \(f''\) → อินทิเกรตหา \(f\)
  • ใช้เงื่อนไขจุดผ่าน \((a, f(a))\) หรือเงื่อนไขอินทิกรัลเพื่อหา \(C\)
  • ระวังเครื่องหมายของ \(\int x^{-n}\,dx = \dfrac{x^{-n+1}}{-n+1}\) เช่น \(\int x^{-2}\,dx = -\dfrac{1}{x}\)

2. Chain Rule + Integration — Q5, Q7, Q11, Q21, Q24, Q26, Q27

  • \((f \circ g)'(x) = f'(g(x)) \cdot g'(x)\)
  • FTC: \(\int_a^b g''(x)\,dx = g'(b) - g'(a)\) — ไม่ต้องหา \(g''\) ทั้งฟังก์ชัน
  • Quotient Rule: \(\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}\)
  • Product Rule: \((uv)' = u'v + uv'\)

3. Area vs Integral — Q2, Q9, Q14, Q19, Q23, Q32, Q35, Q36

  • Integral: มีเครื่องหมาย (บวก-ลบหักล้าง)
  • Area: \(\int_a^b |f(x)|\,dx\) (บวกเสมอ) → ต้องแบ่งช่วงเครื่องหมาย
  • เมื่อ \(f(x)\) เปลี่ยนเครื่องหมาย ณ \(x = c\): แยก \(\int_a^c\) และ \(\int_c^b\)
  • เทคนิคพหุนามดีกรีคี่: เครื่องหมายสลับทุกครั้งที่ข้ามราก

4. Optimization (หาค่าสูงสุด/ต่ำสุด) — Q4, Q6, Q14, Q15, Q18, Q30, Q31

  • หาจุดวิกฤต: \(f'(x) = 0\)
  • ทดสอบ: \(f''(c) > 0\) → ต่ำสุด, \(f''(c) < 0\) → สูงสุด
  • ถ้า \(f' \geq 0\) ทั้งช่วง → \(F\) ไม่ลด → max abs ที่ปลายขวา
  • Optimization ของพื้นที่: แสดงพื้นที่เป็นฟังก์ชันตัวแปรเดียว แล้วดิฟ

5. เส้นสัมผัส (Tangent Line) — Q17, Q27, Q34

  • ความชันของ \(Ax + By + C = 0\) คือ \(-A/B\)
  • เส้นสัมผัสที่ \((a, f(a))\): \(y = f'(a)(x - a) + f(a)\)
  • จุดสัมผัสให้ สอง ข้อมูล: \(f(a)\) และ \(f'(a)\)

6. Piecewise + Periodic — Q16, Q28, Q35

  • แบ่งอินทิเกรตตามจุดเปลี่ยนนิยาม
  • Periodic: หาพื้นที่หนึ่งช่วงแล้วคูณด้วยจำนวนช่วง
  • Substitution \(u = x - n\) ลดรูปการคำนวณ

7. เรขาคณิต — Q29

  • \(\int_{-r}^{r} \sqrt{r^2 - x^2}\,dx = \dfrac{\pi r^2}{2}\) (พื้นที่ครึ่งวงกลม)
  • \(\pi \approx 3.14159\)

8. ตรรกะ \(\forall, \exists\) — Q33

  • \(\forall a[P(a)]\) ผิดเมื่อมี counter-example
  • \(\exists a[P(a)]\) ถูกเมื่อมีอย่างน้อยหนึ่งตัวอย่างที่ทำให้ \(P\) จริง

✅ Checklist สำหรับห้องสอบ — อินทิกรัล

  • อ่านโจทย์: ถาม "พื้นที่" หรือ "อินทิกรัล" — ตัดสินใจว่าจะใช้ \(|f|\) หรือ \(f\) ตรงๆ
  • ระบุฟังก์ชันที่โจทย์ให้: \(f\), \(f'\), \(f''\) หรือ \(\int f\,dx\)
  • ถ้ามี Chain Rule: เขียน \((f \circ g)'\) ออกมาก่อน
  • อินทิเกรตแต่ละขั้นตอนให้ชัดเจน — ระวังเครื่องหมายลบของพจน์ \(x^{-n}\)
  • ใช้ทุกเงื่อนไขที่โจทย์ให้ หาค่าคงตัว \(C_1, C_2, \ldots\)
  • หาจุดวิกฤต \(f'(x) = 0\) แล้วทดสอบ \(f''\)
  • ถ้าพื้นที่: หารากของ \(f\) ในช่วง แล้วแบ่งช่วงเครื่องหมาย
  • เส้นสัมผัส: จำ \(y = f'(a)(x - a) + f(a)\)
  • Optimization: แสดงเป็นฟังก์ชันตัวแปรเดียวก่อนดิฟ
  • หากมี Piecewise: แบ่งอินทิเกรตตามจุดเปลี่ยนนิยาม
  • ตรวจคำตอบเทียบกับ choice ทุกตัว — ถ้าไม่ตรง อาจมีข้อผิดพลาด
  • เช็คเครื่องหมายตอนแทนค่าขอบเขต: \(F(b) - F(a)\) (ลบ!)
  • \(\pi \approx 3.14159\) — คำนวณทศนิยมให้แม่นยำ
  • หากเหลือเวลา: ตรวจ Q2, Q23, Q35 ที่เกี่ยวกับ area/integral ก่อน
  • ตรวจเครื่องหมาย "" (prime) ในโจทย์ — \(f' \neq f\)
