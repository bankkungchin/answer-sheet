ความชันเส้นตรง\(m = \dfrac{y_2-y_1}{x_2-x_1}\)
สมการเส้นตรง (จุด-ความชัน)\(y - y_1 = m(x - x_1)\)
เส้นขนาน\(m_1 = m_2\)
เส้นตั้งฉาก\(m_1 \cdot m_2 = -1\)
ระยะจากจุดถึงเส้นตรง\(d = \dfrac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}\) (เส้น \(Ax+By+C=0\))
จุดกึ่งกลาง\(\left(\dfrac{x_1+x_2}{2},\,\dfrac{y_1+y_2}{2}\right)\)
พื้นที่สามเหลี่ยม (Shoelace)\(S = \dfrac{1}{2}|x_A(y_B-y_C)+x_B(y_C-y_A)+x_C(y_A-y_B)|\)
วงกลม (รูปกลาง)\((x-h)^2+(y-k)^2=r^2\) : ศูนย์กลาง \((h,k)\), รัศมี \(r\)
เส้นตรงตัดวงกลมตัดกัน ⟺ \(d \le r\) · สัมผัส ⟺ \(d = r\) · ไม่ตัด ⟺ \(d > r\)
สี่เหลี่ยมด้านขนานเส้นทแยงมุมแบ่งครึ่งซึ่งกันและกัน → จุดกึ่งกลาง AC = จุดกึ่งกลาง BD

1 Ent ต.ค. 2541 ข้อ 2 อัตนัย (ตอบค่าตัวเลข) ★★★☆☆ 🔴 พื้นที่สามเหลี่ยม
ถ้า \(L_1\) เป็นเส้นตรงที่ผ่านจุด \((-2,\,0)\) และ \((-1,\,2)\) และ \(L_2\) เป็นเส้นตรงที่ผ่านจุดกำเนิดและตั้งฉากกับ \(L_1\) แล้วพื้นที่ของสามเหลี่ยมที่ล้อมรอบด้วยแกน \(x\) เส้นตรง \(L_1\) และเส้นตรง \(L_2\) เท่ากับกี่ตารางหน่วย
💡 แนวคิด
หาสมการเส้นตรงทั้งสอง → หาจุดตัด 3 มุมของสามเหลี่ยม (L1∩แกน x, L2∩แกน x, L1∩L2) → คำนวณพื้นที่ด้วยสูตร Shoelace สะดวกกว่าสูตรฐาน×สูง เพราะไม่ต้องแยกฐานและความสูง
ชั้นที่ 1 หาสมการ L₁

ความชัน \(L_1\): \(\quad m_1 = \dfrac{2-0}{-1-(-2)} = \dfrac{2}{1} = 2\)

สมการ \(L_1\) ผ่าน \((-2,\,0)\) ความชัน 2:

\[y - 0 = 2(x-(-2)) \implies y = 2x+4\]

ชั้นที่ 2 หาสมการ L₂ (ตั้งฉาก L₁, ผ่านจุดกำเนิด)

ความชัน \(L_2 = -\dfrac{1}{m_1} = -\dfrac{1}{2}\)

\(L_2\) ผ่าน \((0,0)\): \(\quad y = -\dfrac{1}{2}x\)

ชั้นที่ 3 หาจุดยอดสามเหลี่ยม A, B, C
  • จุด A = \(L_1\) ∩ แกน \(x\) (ตั้ง \(y=0\)): \(0 = 2x+4 \Rightarrow x = -2\)  →  \(A(-2,\,0)\)
  • จุด B = \(L_2\) ∩ แกน \(x\) (ตั้ง \(y=0\)): \(0 = -\tfrac{1}{2}x \Rightarrow x = 0\)  →  \(B(0,\,0)\)
  • จุด C = \(L_1\) ∩ \(L_2\): \[2x+4 = -\tfrac{1}{2}x \implies \tfrac{5}{2}x = -4 \implies x = -\tfrac{8}{5},\quad y = \tfrac{4}{5}\]  →  \(C\!\left(-\tfrac{8}{5},\,\tfrac{4}{5}\right)\)
ชั้นที่ 4 คำนวณพื้นที่ด้วยสูตร Shoelace

\[S = \tfrac{1}{2}\bigl|x_A(y_B-y_C)+x_B(y_C-y_A)+x_C(y_A-y_B)\bigr|\]

\[= \tfrac{1}{2}\left|(-2)\!\left(0-\tfrac{4}{5}\right)+0\!\left(\tfrac{4}{5}-0\right)+\left(-\tfrac{8}{5}\right)(0-0)\right|\]

\[= \tfrac{1}{2}\left|(-2)\!\left(-\tfrac{4}{5}\right)\right| = \tfrac{1}{2}\cdot\tfrac{8}{5} = \tfrac{4}{5} = \mathbf{0.8}\]

⚠️ จุดพลาดบ่อย — Careless Mistake
กับดัก: สับสนว่า L₂ ผ่านจุด (-2, 0) เหมือน L₁ — แต่โจทย์บอกชัดว่า L₂ ผ่านจุดกำเนิด (0, 0) ดังนั้น B จึงอยู่ที่ (0, 0) ไม่ใช่ (-2, 0)

กับดัก 2: ใช้ (1/2)·ฐาน·สูง แต่หาฐานและสูงผิด — ใช้ Shoelace ปลอดภัยกว่าเมื่อจุดยอดไม่ได้อยู่บนแกน
ตอบ \(0.8\) ตารางหน่วย
2 Ent มี.ค. 2542 ข้อ 3 ปรนัย 4 ตัวเลือก ★★★☆☆ 🟢 เส้นตรงกับวงกลม
ให้ \(S\) เป็นเซตของจำนวนจริง \(m\) ทั้งหมดที่ทำให้เส้นตรง \(y = mx\) ตัดกับวงกลม \(x^2 + y^2 - 10x + 16 = 0\) ขอบเขตบนค่าน้อยสุด (supremum) ของ \(S\) คือข้อใด
\(\dfrac{1}{2}\)
\(\dfrac{2}{3}\)
✓ \(\dfrac{3}{4}\)
\(\dfrac{4}{5}\)
💡 แนวคิด
เส้นตรง \(y = mx\) ตัดวงกลมก็ต่อเมื่อ ระยะจากศูนย์กลางถึงเส้นตรง ≤ รัศมี นำเงื่อนไขนี้มาหา range ของ \(m\) แล้ว supremum คือขอบเขตบนที่น้อยที่สุดของเซต
ชั้นที่ 1 จัดรูปสมการวงกลม

\[x^2 - 10x + y^2 + 16 = 0\]

\[(x-5)^2 - 25 + y^2 + 16 = 0 \implies (x-5)^2 + y^2 = 9\]

ศูนย์กลาง \(O(5,\,0)\), รัศมี \(r = 3\)

ชั้นที่ 2 หาระยะจากศูนย์กลางถึงเส้นตรง

เส้นตรง \(y = mx\) เขียนเป็น \(mx - y = 0\)

ระยะจาก \(O(5,0)\) ถึงเส้น:

\[d = \frac{|5m - 0|}{\sqrt{m^2 + 1}} = \frac{5|m|}{\sqrt{m^2+1}}\]

ชั้นที่ 3 ตั้งอสมการ d ≤ r และแก้หา m

\[\frac{5|m|}{\sqrt{m^2+1}} \le 3\]

ยกกำลังสองทั้งสองข้าง (ทั้งสองข้าง ≥ 0):

\[25m^2 \le 9(m^2+1) \implies 25m^2 - 9m^2 \le 9 \implies 16m^2 \le 9\]

\[m^2 \le \frac{9}{16} \implies |m| \le \frac{3}{4}\]

ดังนั้น \(S = \left[-\dfrac{3}{4},\, \dfrac{3}{4}\right]\)

ชั้นที่ 4 หา supremum ของ S

ขอบเขตบนค่าน้อยสุด (supremum) = ค่าสูงสุดของ \(S\) = \(\dfrac{3}{4}\)

💎 เทคนิคสำหรับแพทย์/วิศวะ
ตรวจคำศัพท์: "ขอบเขตบนค่าน้อยสุด" = Supremum = Least Upper Bound = ค่าสูงสุดของเซต (เมื่อเซตปิด) ถ้าโจทย์ถาม "ขอบเขตล่างค่ามากสุด" = Infimum = Least Lower Bound = ค่าต่ำสุดของเซต
ข้อนี้ \(S = [-3/4, 3/4]\) ดังนั้น \(\sup S = 3/4\) และ \(\inf S = -3/4\)
⚠️ จุดพลาดบ่อย — Careless Mistake
ลืมยกกำลังสองก่อนแก้: ถ้าแก้ \(\dfrac{5|m|}{\sqrt{m^2+1}} \le 3\) โดยย้ายข้างทันทีจะหาร \(\sqrt{m^2+1}\) ผิดพลาดได้ — ต้องยกกำลังสองก่อนเสมอ (ทั้งสองข้างเป็นบวก ดังนั้นสัญลักษณ์ไม่เปลี่ยน)
ตอบ \(\dfrac{3}{4}\)  (ตัวเลือกข้อ 3)
3 Ent ต.ค. 2545 ข้อ 3 ปรนัย 4 ตัวเลือก ★★★★★ 🟠 วงกลมสัมผัสสองกราฟ
กำหนดให้ \(f_1(x) = -\dfrac{x}{2} + \dfrac{3}{2}\) เมื่อ \(x \le 1\) และ \(f_2(x) = 3x - 2\) เมื่อ \(x \ge 1\) ถ้า \(P(a,\,b)\) เป็นจุดศูนย์กลางของวงกลมที่มีรัศมียาว \(\dfrac{7}{\sqrt{5}}\) หน่วย และสัมผัสกราฟของ \(f_1\) และ \(f_2\) แล้ว \(a+b\) เท่ากับข้อใด
\(-2\sqrt{2}\)
\(2\sqrt{2}\)
✓ \(6 - \sqrt{2}\)
\(6 + \sqrt{2}\)
💡 แนวคิด
วงกลมสัมผัสกราฟ (เส้นตรง) ⟺ ระยะจากศูนย์กลาง P(a,b) ถึงแต่ละเส้นตรง = รัศมี แปลงกราฟทั้งสองเป็นสมการเส้นตรง จากนั้นตั้งระบบสมการ 2 สมการ ค่าสัมบูรณ์ → เลือกเครื่องหมายด้วย "เทคนิค Side-of-Line"
ชั้นที่ 1 แปลงกราฟเป็นสมการเส้นตรงรูปมาตรฐาน

\(f_1:\; y = -\dfrac{x}{2}+\dfrac{3}{2}\)  →  คูณสองข้างด้วย 2:  \(x + 2y - 3 = 0\)  [\(A_1=1,B_1=2,C_1=-3\)]

\(f_2:\; y = 3x-2\)  →  \(3x - y - 2 = 0\)  [\(A_2=3,B_2=-1,C_2=-2\)]

ตรวจสอบ: ทั้งสองกราฟตัดกันที่จุด \((1,1)\) ✓

ชั้นที่ 2 ตั้งสมการระยะ = รัศมี

ระยะจาก \(P(a,b)\) ถึงเส้น \(f_1: x+2y-3=0\):

\[\frac{|a+2b-3|}{\sqrt{1^2+2^2}} = \frac{7}{\sqrt{5}} \implies \frac{|a+2b-3|}{\sqrt{5}} = \frac{7}{\sqrt{5}} \implies |a+2b-3| = 7 \quad\cdots(*)\]

ระยะจาก \(P(a,b)\) ถึงเส้น \(f_2: 3x-y-2=0\):

\[\frac{|3a-b-2|}{\sqrt{3^2+1^2}} = \frac{7}{\sqrt{5}} \implies \frac{|3a-b-2|}{\sqrt{10}} = \frac{7}{\sqrt{5}} \implies |3a-b-2| = \frac{7\sqrt{10}}{\sqrt{5}} = 7\sqrt{2} \quad\cdots(**)\]

💎 เทคนิค Side-of-Line — ตัดค่าสัมบูรณ์ไม่ต้องแยก 4 กรณี
สองกราฟตัดกันที่ \((1,1)\) สร้างมุม 4 ส่วน วงกลมอยู่ใน ส่วนเดียว ทดสอบจุดภายในมุม เช่น \((0,3)\):
  • \(L_1\): \(0+2(3)-3 = 3 > 0\) → จุดนี้อยู่ฝั่ง บวก ของ \(L_1\) → วงกลมในฝั่งนี้ใช้เครื่องหมาย +7
  • \(L_2\): \(3(0)-3-2 = -5 < 0\) → จุดนี้อยู่ฝั่ง ลบ ของ \(L_2\) → วงกลมในฝั่งนี้ใช้เครื่องหมาย −7√2
ดังนั้นเลือก: \(a+2b-3 = +7\) และ \(3a-b-2 = -7\sqrt{2}\) ให้คำตอบที่สอดคล้องกับตัวเลือก
ชั้นที่ 3 แก้ระบบสมการ (เลือกเครื่องหมาย + และ −)

จาก \((*)\): \(a + 2b - 3 = 7 \implies a + 2b = 10 \quad\cdots(1)\)

จาก \((**)\): \(3a - b - 2 = -7\sqrt{2} \implies 3a - b = 2 - 7\sqrt{2} \quad\cdots(2)\)

นำ (1) + 2×(2):

\[(a+2b) + (6a-2b) = 10 + 2(2-7\sqrt{2})\]

\[7a = 10 + 4 - 14\sqrt{2} = 14 - 14\sqrt{2} \implies a = 2 - 2\sqrt{2}\]

จาก (1): \(b = \dfrac{10-a}{2} = \dfrac{10-(2-2\sqrt{2})}{2} = \dfrac{8+2\sqrt{2}}{2} = 4+\sqrt{2}\)

ชั้นที่ 4 คำนวณ a + b

\[a + b = (2 - 2\sqrt{2}) + (4 + \sqrt{2}) = 6 - \sqrt{2}\]

⚠️ จุดพลาดบ่อย — Wrong Concept
ข้อผิดพลาดของ DeepSeek (เฉลยอ้างอิง): ใช้ \(f_1(x) = -x+3\) (ผิด) และรัศมี \(7/5\) (ผิด) สมการที่ถูกต้องคือ \(f_1(x) = -x/2 + 3/2\) ซึ่งทำให้ทั้งสองกราฟตัดกันที่ \((1,1)\) ได้จริง และรัศมีที่ถูกคือ \(7/\sqrt{5}\) — ถ้าใช้ค่าผิดจะตั้งระบบสมการผิดและไม่ได้ \(6-\sqrt{2}\)

กับดัก: เลือกเครื่องหมายผิดในค่าสัมบูรณ์ → ตรวจสอบด้วยเทคนิค Side-of-Line ทุกครั้ง
ตอบ \(6 - \sqrt{2}\)  (ตัวเลือกข้อ 3)
4 Ent มี.ค. 2546 ข้อ 3 ปรนัย 4 ตัวเลือก ★★★☆☆ 🔵 เส้นตั้งฉากแบ่งครึ่ง
กำหนดให้ \(a\) เป็นจำนวนจริง และ \(A(a,\,1)\), \(B(-5,\,-4)\), \(C(1,\,-2)\), \(D(2,\,3)\) เป็นจุดยอดของรูปสี่เหลี่ยมด้านขนาน \(ABCD\) ถ้า \(L\) เป็นเส้นตรงที่ตั้งฉากกับ \(AC\) และผ่านจุดกึ่งกลางของ \(AC\) แล้วสมการของเส้นตรง \(L\) คือข้อใด
✓ \(5x - 3y + 6 = 0\)
\(5x - 3y - 6 = 0\)
\(5x + 3y + 9 = 0\)
\(5x + 3y - 9 = 0\)
💡 แนวคิด
สี่เหลี่ยมด้านขนาน → เส้นทแยงมุม AC และ BD แบ่งครึ่งซึ่งกันและกัน ดังนั้นจุดกึ่งกลาง AC = จุดกึ่งกลาง BD → นำไปหาค่า \(a\) จากนั้น L เป็นเส้นตั้งฉากกับ AC ที่ผ่านจุดกึ่งกลางของ AC (Perpendicular Bisector)
ชั้นที่ 1 หาค่า a จากสมบัติสี่เหลี่ยมด้านขนาน

จุดกึ่งกลาง \(BD\): \(\left(\dfrac{-5+2}{2},\,\dfrac{-4+3}{2}\right) = \left(-\dfrac{3}{2},\,-\dfrac{1}{2}\right)\)

จุดกึ่งกลาง \(AC\): \(\left(\dfrac{a+1}{2},\,\dfrac{1+(-2)}{2}\right) = \left(\dfrac{a+1}{2},\,-\dfrac{1}{2}\right)\)

เนื่องจากเป็นจุดเดียวกัน (y-พิกัดตรงกันอยู่แล้ว):

\[\frac{a+1}{2} = -\frac{3}{2} \implies a+1 = -3 \implies a = -4\]

ดังนั้น \(A = (-4,\,1)\)

ชั้นที่ 2 หาความชันของ AC

\(A(-4,\,1)\) และ \(C(1,\,-2)\)

\[m_{AC} = \frac{-2-1}{1-(-4)} = \frac{-3}{5} = -\frac{3}{5}\]

ชั้นที่ 3 หาความชันของ L (ตั้งฉาก AC)

\[m_L = -\frac{1}{m_{AC}} = -\frac{1}{-3/5} = \frac{5}{3}\]

ชั้นที่ 4 หาสมการ L ผ่านจุดกึ่งกลาง AC

จุดกึ่งกลาง \(AC = \left(-\dfrac{3}{2},\,-\dfrac{1}{2}\right)\)

สมการ \(L\): \(y - \left(-\dfrac{1}{2}\right) = \dfrac{5}{3}\left(x - \left(-\dfrac{3}{2}\right)\right)\)

\[y + \frac{1}{2} = \frac{5}{3}\!\left(x + \frac{3}{2}\right)\]

คูณทั้งสองข้างด้วย 6:

\[6y + 3 = 10x + 15 \implies 10x - 6y + 12 = 0 \implies \boxed{5x - 3y + 6 = 0}\]

💎 เทคนิคตรวจสอบ
แทนจุดกึ่งกลาง \((-3/2, -1/2)\) ลงในสมการ: \(5(-3/2) - 3(-1/2) + 6 = -15/2 + 3/2 + 6 = -6 + 6 = 0\) ✓ ตรวจความชัน: \(5x - 3y + 6 = 0 \Rightarrow y = 5x/3 + 2\), ความชัน \(= 5/3\) ✓ และ \(m_{AC} \cdot m_L = (-3/5)(5/3) = -1\) ✓ (ตั้งฉากกัน)
⚠️ จุดพลาดบ่อย — Careless Mistake
ลืมหาค่า \(a\) ก่อน แล้วไปใช้ \(A(a, 1)\) โดยไม่แทนค่า → ความชัน AC จะผิด
ต้องทำ ชั้นที่ 1 ก่อนเสมอ: หา \(a\) จากสมบัติสี่เหลี่ยมด้านขนาน แล้วค่อยหาความชัน
ตอบ \(5x - 3y + 6 = 0\)  (ตัวเลือกข้อ 1)
