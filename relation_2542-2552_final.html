Ent มี.ค.42 ★★☆☆☆ 🔵 โดเมนและเรนจ์ ตอบ: ข้อ 2
กำหนดให้ \(r\) เป็นความสัมพันธ์ในเซตของจำนวนจริง โดยที่ \[r = \left\{(x,y)\,\middle|\, y = \sqrt{\dfrac{1-x^2}{1+x^2}}\right\}\] ข้อใดต่อไปนี้ถูกต้อง?
1) \(D_r = [-1,1],\ D_{r^{-1}} = [-1,1]\)
2) \(D_r = [-1,1],\ D_{r^{-1}} = [0,1]\)
3) \(D_r = [0,1],\ D_{r^{-1}} = [-1,1]\)
4) \(D_r = [0,1],\ D_{r^{-1}} = [0,1]\)
💡 แนวคิด
\(D_{r^{-1}} = R_r\) และ \(R_{r^{-1}} = D_r\) เสมอ (การผกผันสลับโดเมนกับเรนจ์)
Step 1 หา \(D_r\): ต้องการ \(\dfrac{1-x^2}{1+x^2} \ge 0\). เนื่องจาก \(1+x^2 > 0\) เสมอ จึงต้องการ \(1-x^2 \ge 0\) → \(x^2 \le 1\) → \(x \in [-1,1]\). ดังนั้น \(D_r = [-1,1]\)
Step 2 หา \(R_r\): สำหรับ \(x \in [-1,1]\): ที่ \(x=0\): \(y = \sqrt{1/1} = 1\) (สูงสุด). ที่ \(x = \pm 1\): \(y = \sqrt{0/2} = 0\) (ต่ำสุด). ดังนั้น \(R_r = [0,1]\)
Step 3 หา \(D_{r^{-1}}\): \(D_{r^{-1}} = R_r = [0,1]\)
⚠️ จุดระวัง
อย่าสับสนว่า \(D_{r^{-1}} = D_r\) — จริงๆ แล้ว \(D_{r^{-1}} = R_r\)!
เฉลย ข้อ 2: \(D_r = [-1,1],\ D_{r^{-1}} = [0,1]\) ✓

ก. \(D_r = (-\infty,-1)\cup(1,\infty)\)
ข. \(r^{-1} = \{(x,y)\mid y = \pm\sqrt{\dfrac{1+x}{x}}\}\)
ข้อใดถูกต้อง?

⚠️ จุดระวัง
\(r^{-1}\) ต้องใช้ \(|x|\) เพราะโดเมนของ \(r^{-1}\) (= เรนจ์ของ \(r\)) มีทั้ง \(x>0\) และ \(x<0\) ในฝั่งบวก/ลบ

1) \(\left(-\dfrac{1}{3},\dfrac{1}{2}\right]\)  2) \(\left[-\dfrac{1}{2},\dfrac{1}{3}\right)\)  3) \(\left(-\infty,-\dfrac{1}{3}\right)\cup\left(-\dfrac{1}{3},\infty\right)\)  4) \((-\infty,\infty)\)
💡 แนวคิด
\(R_{r^{-1}} = D_r\) ดังนั้นเราต้องหาโดเมนของ \(r\) ก่อน
Step 1 จัดรูป: \(2x^3 - x^2 + y^2(3x+1) = 0\) → \(y^2 = \dfrac{x^2-2x^3}{3x+1} = \dfrac{x^2(1-2x)}{3x+1}\)
Step 2 เงื่อนไข: ต้องการ \(y^2 \ge 0\) และ \(3x+1 \ne 0\) (คือ \(x \ne -\tfrac{1}{3}\)). เนื่องจาก \(x^2 \ge 0\) เสมอ จึงต้องการ \(\dfrac{1-2x}{3x+1} \ge 0\)
Step 3 วิเคราะห์เครื่องหมาย: จุดวิกฤต \(x = -\tfrac{1}{3}\) (ตัวส่วน) และ \(x = \tfrac{1}{2}\) (ตัวเศษ)

  • \(-\tfrac{1}{3} < x < \tfrac{1}{2}\): ทั้งคู่บวก → ผลบวก ✓
  • \(x = \tfrac{1}{2}\): \(y^2 = 0\) ✓
  • \(x > \tfrac{1}{2}\): \((1-2x)<0\), \((3x+1)>0\) → ผลลบ ✗
Step 4 สรุป: \(D_r = \left(-\dfrac{1}{3},\dfrac{1}{2}\right]\) → \(R_{r^{-1}} = \left(-\dfrac{1}{3},\dfrac{1}{2}\right]\)
⚠️ จุดระวัง
\(x = -\tfrac{1}{3}\) ต้องยกเว้น (ตัวส่วน = 0) → ใช้วงเล็บเปิดที่ปลายซ้าย. แต่ \(x = \tfrac{1}{2}\) ทำให้ \(y^2 = 0\) ได้จริง → ใช้วงเล็บปิดที่ปลายขวา
เฉลย ข้อ 1: \(\left(-\dfrac{1}{3},\dfrac{1}{2}\right]\) ✓

A-NET ก.พ.49 ★★★☆☆ 🟣 ผลคูณคาร์ทีเซียน ตอบ: 15
กำหนดให้ \(A = \{1,\, 2,\, \{1,2\},\, (1,2)\}\) เมื่อ \((1,2)\) หมายถึงคู่อันดับ และ \(B = (A\times A) - A\)
จำนวนสมาชิกของเซต \(B\) เท่ากับเท่าใด?
💡 แนวคิด
\(B = (A\times A) - A\) ลบเฉพาะสมาชิกของ \(A\) ที่อยู่ใน \(A\times A\) ด้วย. ตัวเลข 1, 2 และเซต \(\{1,2\}\) ไม่ใช่คู่อันดับ จึงไม่อยู่ใน \(A\times A\)
Step 1 นับสมาชิก \(A\): \(|A| = 4\) (สมาชิก 4 ตัว: \(1\), \(2\), \(\{1,2\}\), \((1,2)\))
Step 2 ขนาด \(A\times A\): \(|A \times A| = 4^2 = 16\)
Step 3 หา \(A \cap (A\times A)\): สมาชิกของ \(A\times A\) คือคู่อันดับทั้งหมด. สมาชิกของ \(A\) ที่เป็นคู่อันดับมีแค่ \((1,2)\). ตรวจ: \(1 \in A\) และ \(2 \in A\) ดังนั้น \((1,2) \in A\times A\) ✓. ดังนั้น \(A \cap (A\times A) = \{(1,2)\}\), \(|A \cap (A\times A)| = 1\)
Step 4 คำนวณ \(|B|\): \(|B| = |A\times A| - |A \cap (A\times A)| = 16 - 1 = 15\)
เฉลย 15 ✓

เซตในข้อใดต่อไปนี้เป็นสับเซตของ \(D_r - D_s\)?
1) \([-4,-1]\)  2) \([-3,0]\)  3) \([-2,1]\)  4) \([-1,2]\)
Step 1 หา \(D_r\): จาก \(x^2+y^2=16\): ต้องการ \(y^2 = 16-x^2 \ge 0\) → \(|x| \le 4\) → \(D_r = [-4,4]\)
Step 2 หา \(D_s\): จัดรูป \(xy^2+x+3y^2+2=0\) → \(y^2(x+3) = -x-2\) → \(y^2 = \dfrac{-x-2}{x+3} = \dfrac{-(x+2)}{x+3}\).
ต้องการ \(y^2 \ge 0\) และ \(x \ne -3\): \(\dfrac{-(x+2)}{x+3} \ge 0\) → \(\dfrac{x+2}{x+3} \le 0\) → \(-3 < x \le -2\).
ดังนั้น \(D_s = (-3,-2]\)
Step 3 หา \(D_r - D_s\): \(D_r - D_s = [-4,4] \setminus (-3,-2] = [-4,-3] \cup (-2,4]\)
Step 4 ตรวจตัวเลือก:
  • 1) \([-4,-1]\): มี \(-2 \in (-3,-2]\) → ไม่เป็นสับเซต ✗
  • 2) \([-3,0]\): มี \(-2.5 \in (-3,-2]\) → ไม่เป็นสับเซต ✗
  • 3) \([-2,1]\): \(-2 \in (-3,-2]\) → ไม่เป็นสับเซต ✗
  • 4) \([-1,2]\): ทุกค่าใน \([-1,2] \subset (-2,4] \subset D_r - D_s\) ✓
✅ เทคนิค
จัดรูป \(s\) เป็น \(y^2 = f(x)\) แล้วหาเงื่อนไข \(f(x) \ge 0\) → \(D_s\)
เฉลย ข้อ 4: \([-1,2]\) ✓

A-NET มี.ค.51 ★★★★☆ 🔵 โดเมนและเรนจ์ ตอบ: ข้อ 3
กำหนดให้ \(r = \{(x,y)\mid (x-2)(y-1)=1\}\), \(s = \{(x,y)\mid xy^2=(y+1)^2\}\)
เซตในข้อใดต่อไปนี้ ไม่เป็นสับเซตของ \(R_r \cap R_s\)?
1) \((-\infty,-1)\)  2) \(\left(-2,-\dfrac{1}{2}\right)\)  3) \(\left(\dfrac{1}{2},2\right)\)  4) \((1,\infty)\)
Step 1 หา \(R_r\): จาก \((x-2)(y-1)=1\) → \(y-1 = \dfrac{1}{x-2}\) → \(y = 1 + \dfrac{1}{x-2}\). เนื่องจาก \(\dfrac{1}{x-2} \ne 0\) เสมอ → \(y \ne 1\). ดังนั้น \(R_r = \mathbb{R}\setminus\{1\}\)
Step 2 หา \(R_s\): จาก \(xy^2 = (y+1)^2\). ที่ \(y=0\): \(0 = 1\) ขัดแย้ง → \(y=0\) ไม่อยู่ใน \(R_s\). สำหรับ \(y \ne 0\): \(x = \dfrac{(y+1)^2}{y^2} \ge 0\) เสมอ (เป็นจำนวนจริงที่ไม่เป็นลบ) → ทุก \(y \ne 0\) มี \(x\) ที่สอดคล้อง. ดังนั้น \(R_s = \mathbb{R}\setminus\{0\}\)
Step 3 หา \(R_r \cap R_s\): \((\mathbb{R}\setminus\{1\}) \cap (\mathbb{R}\setminus\{0\}) = \mathbb{R}\setminus\{0,1\}\)
Step 4 ตรวจตัวเลือก:
  • 1) \((-\infty,-1)\): ไม่มี 0 หรือ 1 อยู่ภายใน → เป็นสับเซต ✓
  • 2) \((-2,-\tfrac{1}{2})\): ไม่มี 0 หรือ 1 อยู่ภายใน → เป็นสับเซต ✓
  • 3) \((\tfrac{1}{2},2)\): มี \(1 \in (\tfrac{1}{2},2)\) แต่ \(1 \notin \mathbb{R}\setminus\{0,1\}\) → ไม่เป็นสับเซต ✗
  • 4) \((1,\infty)\): 1 ไม่ถูกรวม (ช่วงเปิด), 0 ไม่อยู่ในช่วง → เป็นสับเซต ✓
⚠️ จุดระวัง
\(\left(\tfrac{1}{2},2\right)\) มี \(1\) อยู่ด้านใน (1 คือจุดภายในของช่วงเปิด) → \(1 \notin R_r \cap R_s\) → ช่วงนี้ไม่เป็นสับเซต
เฉลย ข้อ 3: \(\left(\dfrac{1}{2},2\right)\) ✓

A-NET มี.ค.51 ★★★☆☆ 🔴 เซตในระนาบ ตอบ: ข้อ 4
กำหนดให้ \(A = \{(x,y)\mid x^2+y^2>1\}\), \(B = \{(x,y)\mid 4x^2+9y^2<1\}\), \(C = \{(x,y)\mid y^2-x^2>1\}\)
ข้อใดต่อไปนี้ ผิด?
1) \(A-B=A\)  2) \(B-C=B\)  3) \(B\cap(A\cup C)=\emptyset\)  4) \(A\cap(B\cup C)=\emptyset\)
Step 1 ความสัมพันธ์ระหว่างเซต:
  • \(B \cap A = \emptyset\): ถ้า \(4x^2+9y^2<1\) แล้ว \(x^2+y^2 \le 4x^2+9y^2 < 1\) → \(x^2+y^2 < 1\) → \((x,y) \notin A\) → \(B \subset A^c\)
  • \(C \subset A\): ถ้า \(y^2-x^2>1\) แล้ว \(y^2 > x^2+1 \ge 1\) → \(x^2+y^2 > x^2+1 \ge 1\) → \((x,y) \in A\)
  • \(B \cap C = \emptyset\): \(B \subset A^c\) แต่ \(C \subset A\) → \(B \cap C = \emptyset\)
Step 2 ตรวจข้อ:
  • 1) \(A-B = A\): เพราะ \(B \cap A = \emptyset\) → ถูก ✓
  • 2) \(B-C = B\): เพราะ \(B \cap C = \emptyset\) → ถูก ✓
  • 3) \(B\cap(A\cup C) = (B\cap A)\cup(B\cap C) = \emptyset\cup\emptyset = \emptyset\) → ถูก ✓
  • 4) \(A\cap(B\cup C) = (A\cap B)\cup(A\cap C) = \emptyset \cup C = C \ne \emptyset\) → ผิด ✗
✅ เทคนิค
วาด Venn diagram: วงรีเล็ก \(B\) (ภายในวงรี \(4x^2+9y^2=1\)) อยู่นอกวงกลม \(A\); บริเวณ \(C\) (ส่วน hyperbola \(y^2-x^2>1\)) อยู่ใน \(A\)
เฉลย ข้อ 4: \(A\cap(B\cup C)=\emptyset\) เป็นข้อที่ผิด ✓

A-NET 2552 ★★★★☆ 🔵 โดเมนและเรนจ์ ตอบ: ข้อ 2
กำหนดให้ \(r = \left\{(x,y)\,\middle|\, x>0,\; x\ne y,\; x - \sqrt[3]{x} = y - \sqrt[3]{y}\right\}\)
สมาชิกค่ามากที่สุดของ \(D_r\) เท่ากับข้อใด?
1) \(\dfrac{4}{3\sqrt{3}}\)  2) \(\dfrac{8}{3\sqrt{3}}\)  3) \(\dfrac{4}{9}\)  4) \(\dfrac{8}{9}\)
💡 แนวคิด
แทน \(u = \sqrt[3]{x},\; v = \sqrt[3]{y}\) แล้วใช้ \(u \ne v\) เพื่อหาร → \(u^2+uv+v^2=1\) → discriminant ≥ 0 → ขอบเขต \(u\)
Step 1 แทนตัวแปร: ให้ \(u = \sqrt[3]{x},\; v = \sqrt[3]{y}\) (โดย \(u > 0\)): \[u^3 - u = v^3 - v \implies u^3 - v^3 = u - v \implies (u-v)(u^2+uv+v^2) = (u-v)\]
Step 2 ลดรูป: เนื่องจาก \(x \ne y\) → \(u \ne v\) → หารทั้งสองข้างด้วย \((u-v)\): \[\boxed{u^2 + uv + v^2 = 1}\]
Step 3 หาขอบเขต \(u\): มองเป็นสมการกำลังสองใน \(v\): \(v^2 + (u)v + (u^2-1) = 0\)
\[v = \frac{-u \pm \sqrt{u^2 - 4(u^2-1)}}{2} = \frac{-u \pm \sqrt{4-3u^2}}{2}\] ต้องการ \(v\) เป็นจำนวนจริง: \(4-3u^2 \ge 0\) → \(u^2 \le \dfrac{4}{3}\) → \(u \le \dfrac{2}{\sqrt{3}}\)
Step 4 ตรวจ \(u = \tfrac{2}{\sqrt{3}}\): \(v = \dfrac{-2/\sqrt{3} \pm 0}{2} = -\dfrac{1}{\sqrt{3}} \ne u\) ✓ (เงื่อนไข \(u \ne v\) สอดคล้อง)
Step 5 หา \(x_{\max}\): \[x_{\max} = u_{\max}^3 = \left(\frac{2}{\sqrt{3}}\right)^3 = \frac{8}{3\sqrt{3}}\]
⚠️ จุดระวัง
โจทย์ใช้ รากที่สาม (\(\sqrt[3]{x}\)) ไม่ใช่รากที่สอง — ต้องแทน \(u = \sqrt[3]{x}\) ไม่ใช่ \(u = \sqrt{x}\)!
เฉลย ข้อ 2: \(\dfrac{8}{3\sqrt{3}}\) ✓
